NoobSoft - Noob Console controller inputs

Action Buttons

  • controller.a = +
  • controller.b = -
  • controller.plus = +
  • controller.dash = -
  • controller.slash = /

Direction

  • controller.up = Up
  • controller.down = Down
  • controller.left = Left
  • controller.right = Right

Additional Action Buttons (3D Controller)

  • controller.multiply = *
  • controller.select = ()
  • controller.backslash = \

Secondary Directions (3D Controller)

  • controller.up2 = Secondary Up
  • controller.down2 = Secondary Down
  • controller.left2 = Secondary Left
  • controller.right2 = Secondary Right
